Bookings

Shoot days booked via your Google Calendar appointment page.

Setup checklist
✓
Step 1 — Google Calendar appointment page
calendar.google.com → + Create → Appointment schedule. Add custom questions: Company, Shoot type, Budget. Copy the booking URL for your Instagram bio.
○
Step 2 — Set Railway environment variables
In Railway → Variables, add: GOOGLE_WEBHOOK_TOKEN (any random string), GOOGLE_CALENDAR_ID (your Gmail address), RAILWAY_PUBLIC_DOMAIN (ogm-outreach-production.up.railway.app)
○
Step 3 — Upload service account key
Go to console.cloud.google.com → APIs & Services → Enable Calendar API → Credentials → Create Service Account → download the JSON key → upload it to your Railway project as /app/google-service-account.json using Railway's file mount feature.
○
Step 4 — Register the webhook (once)
After deploying, open your browser and go to: https://ogm-outreach-production.up.railway.app/api/calendar/register (POST request — use a tool like Postman or the button below).
